# UTT Nodo Morón - Plataforma de Consumo Consciente

Prototipo funcional para la digitalización del Nodo Morón de la Unión de Trabajadores de la Tierra. 

## 🚀 Características
- **UX Optimizada:** Navegación fluida que no bloquea al usuario.
- **Sistema de Reparto Inteligente:** Validación obligatoria de Localidad, Calle y Altura solo al momento de la compra.
- **Mobile First:** Diseñado para ser utilizado cómodamente desde celulares.
- **Estética Orgánica:** Interfaz alineada con los valores de la agroecología.

## 🛠️ Tech Stack
- HTML5 / Tailwind CSS
- Lucide Icons
- JavaScript (Lógica de validación de logística)

## 📋 Próximos Pasos
- Conexión con base de datos (Supabase) para control de stock en tiempo real.
- Panel de administración para el Nodo.
- Integración automatizada con WhatsApp.
